# Golestan Website (React + Tailwind)

Home page implementation for the Golestan website, built with **React** and **Tailwind CSS**.  
This repository currently contains the **homepage UI**, and it is designed to be extended with additional sections and pages over time.

## Live Demo
[![Netlify Status](https://api.netlify.com/api/v1/badges/a08eeb9a-4a35-439d-9470-e348d19ea3cb/deploy-status)](https://app.netlify.com/projects/golestan-react/deploys)
---

## Tech Stack

- **React**
- **Tailwind CSS**
- Component-based UI architecture
- Responsive layout (mobile-first)

---

## Features

- Modern, responsive homepage layout
- Reusable UI components (Navbar, Mobile Menu, sections, cards, etc.)
- Clean Tailwind class organization (constants for classNames)
- Ready for scaling into a multi-page website

---

## Project Structure (Overview)

Typical structure:

- `src/components/` – Reusable components (Navbar, MobileMenu, sections, UI base components)
- `src/assets/` – Images, icons, static assets
- `src/pages/`  – Pages / routes as the project grows
